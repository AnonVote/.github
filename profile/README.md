<h1>AnonVote</h1>

<p>Private decision infrastructure for organizations on Stellar.</p>

</div>

---

AnonVote is a privacy-preserving voting platform that separates voter identity from ballots at every layer — cryptographically, not by policy. Results are anchored to the Stellar blockchain so anyone can verify outcomes without trusting AnonVote's servers.

## Repositories

| Repo | Description |
|------|-------------|
| [`core`](https://github.com/AnonVote/core) | Express backend, React frontend, Prisma + PostgreSQL |
| [`js`](https://github.com/AnonVote/js) | `@anonvote/crypto` — shared crypto primitives and TypeScript types |
| [`contracts`](https://github.com/AnonVote/contracts) | Soroban smart contracts in Rust |
| [`protocol`](https://github.com/AnonVote/protocol) | Whitepaper, specs, and token flow documentation |
